# Security Disclaimer

**⚠️ Important Security Considerations**

## AI-Generated Code Execution

- The playground UI is **generated by AI** and executes in your browser
- AI may produce code with bugs, security vulnerabilities, or unexpected behavior
- **Always review** AI-generated code before using in sensitive environments

## File System Access

- The AI CLI tool has access to your file system through the workspace directories
- It can read, write, and execute files within the project
- Run Freeflow in an isolated environment for untrusted AI interactions

## Command Execution

- AI agents can execute shell commands through the terminal
- Commands run with your user permissions
- **Review all commands** before allowing execution, especially those involving:
  - `rm`, `sudo`, `curl | bash`, or package installations
  - Network requests to external services
  - File system modifications outside the project

## Recommended Practices

- Use a dedicated workspace for AI experiments
- Keep backups of important code
- Run in a container or VM for additional isolation
- Do not expose the Freeflow server to public networks
